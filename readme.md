# ⚖️ Hukum Jalanan

![Unreal Engine](https://img.shields.io/badge/Unreal%20Engine-5.1.1-0E1128?style=for-the-badge&logo=unrealengine&logoColor=white)
![Status](https://img.shields.io/badge/Status-Development-orange?style=for-the-badge)

**Hukum Jalanan** adalah sebuah proyek game horror-adventure yang dikembangkan menggunakan **Unreal Engine 5.1.1**. Game ini fokus pada atmosfer yang mencekam, narasi yang kuat, dan eksplorasi lingkungan yang detail.

## 📝 Deskripsi Proyek
Game ini menceritakan tentang [masukkan sedikit sinopsis cerita di sini, misal: perjalanan seorang karakter mencari kebenaran di jalanan yang penuh misteri]. Proyek ini juga merupakan eksplorasi teknis dalam penggunaan:
* **Unreal Engine Blueprint & C++**
* **Git LFS** untuk manajemen aset besar.
* **High-quality environment design.**

## 🛠️ Tech Stack
* **Engine:** Unreal Engine 5.1.1
* **Version Control:** Git + Git LFS
* **Assets:** Custom Blueprints, Quixel Bridge, Modular Medieval Town with Interior, Modular Old Town, Downtown Alley, City Street Props.

## 📂 Struktur Proyek
* `Content/` - Berisi semua aset game (Blueprints, Meshes, Textures, Maps).
* `Config/` - Konfigurasi project dan input settings.
* `Hukum_Jalanan.uproject` - File utama untuk membuka proyek.

## 🚀 Cara Menjalankan (Development)
1. Pastikan kamu sudah menginstal **Git LFS** (`git lfs install`).
2. Clone repository ini:
   ```bash
   git clone [https://github.com/FarrelApriandry/Hukum-Jalanan.git](https://github.com/FarrelApriandry/Hukum-Jalanan.git)
    ```
3. Klik kanan pada `Hukum_Jalanan.uproject` dan pilih **Generate Visual Studio project files** (jika menggunakan C++).
4. Buka proyek di Unreal Engine 5.1.1.
